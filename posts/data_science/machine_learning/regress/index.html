4. 표준화잔차, cook’s distance, DFBeta, 공분산비 보기

  • 표준화잔차(standard_resid) : 잔차를 표준오차로 나눈 값
  • cook’s distance : 관측치가 제거되었을 때 모수 추정값들의 (동시적) 변화량 척도
  • DFBeta : 관측값이 각각의 회귀계수에 미치는 영향력을 측정
  • DFFITS : 관측치가 예측값에서 보유하고 있는 영향력을 측정
